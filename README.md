# Asset Tracker Frontend

資産管理アプリのフロントエンド（React + JavaScript）です。

## 公開URL

👉 [アプリはこちら](https://asset-tracker-frontend.vercel.app/)

## 主な使用技術

- React
- JavaScript
- Recharts（グラフ描画）
- CSS（レスポンシブ対応済み）
- Vercel（ホスティング）

## セットアップ

```bash
npm install
npm start
```

## 主な機能

```md
- ログイン／新規登録（ユーザー認証）
- 初期資産の登録（1回のみ）
- 月ごとの収入・支出入力
- 資産の推移をグラフで可視化（棒グラフ／折れ線グラフの切り替え対応）
- モバイルでも見やすいレスポンシブ対応UI
```
